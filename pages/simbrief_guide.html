Flight Planning with SimBrief

Learn how to create professional flight plans and log your flights using our Discord bot

1

Create a SimBrief Account

If you don't already have one, sign up for a free account at SimBrief.com. This is essential for creating detailed flight plans.

Important: Make sure your account is set to public to allow the Discord bot to fetch your flight plans.

2

Create Your Flight Plan

Navigate to the Dispatch page and enter your flight details:

  • Departure Airport: Your starting airport (e.g., NZCH)
  • Arrival Airport: Your destination airport
  • Aircraft: Select from our fleet or use a similar type
  • Route: Enter your planned route or use "Direct"
  • Passengers: Set according to your charter mission
  • Weather Source: Use any source you prefer
  • Options: Enable "Generate PDF" for full OFP access
3

Find Your SimBrief User ID

You'll need your numeric SimBrief User ID (not your username):

1. Log into SimBrief and click your username in the top right
2. Select "Account Settings" from the dropdown
3. Look for "User ID" in the Account Information section
4. It's a numeric value like: 123456

Alternative: When viewing a flight plan, check the URL. If it looks like https://www.simbrief.com/ofp/uads/?userid=123456, your ID is 123456.

4

Use the Discord Bot Command

Once your flight plan is created, share it with the community using our custom Discord bot:

Command Format:
/simbrief-pirep [Your SimBrief User ID]

Example: /simbrief-pirep 123456

Permission Requirements

The /simbrief-pirep command requires one of the following CPAS roles:

Pilot Under Training
Charter Pilot
Senior Charter Pilot
Lead Pilot
Instructor
Administrators

If you don't have permission, you'll receive an error message and the command won't process.

Required Information

Departure & Arrival

Must include both airports with proper ICAO codes (e.g., NZCH, NZQN)

Fuel Planning

Ensure adequate fuel reserves (45 minutes minimum for CPAS operations)

Passenger Count

Set realistic passenger numbers based on aircraft capacity

Route Details

Include waypoints, airways, or use "Direct" for VFR charters

Weather Consideration

Check weather and NOTAMs before finalizing your plan

Aircraft Configuration

Select correct aircraft type and configuration for your route

Pro Tips

Save Your Plans

Export to MSFS

Use the "Export" feature to send your flight plan directly to Microsoft Flight Simulator

Mobile Friendly

Track Progress

Debugging Help

If the command fails, try manually visiting: https://www.simbrief.com/api/xml.fetcher.php?userid=YOUR_ID to see if it returns valid XML.

Common Issues & Solutions

  • Permission Denied: Ensure you have a CPAS pilot role assigned in Discord
  • Invalid Pilot ID: Check your SimBrief Account Settings for your numeric User ID
  • No Flight Plan Found: The bot only fetches your latest flight plan - create one first!
  • Private Profile: Your SimBrief profile must be set to public for the bot to access your flight plans
  • XML Parsing Error: Sometimes occurs with incomplete or corrupted flight plans
  • Rate Limiting: SimBrief API has rate limits - wait a few minutes before retrying
  • Incomplete Fields: Ensure all required fields are filled in your SimBrief flight plan

Technical Details

  • The bot uses SimBrief's XML API: https://www.simbrief.com/api/xml.fetcher.php?userid=YOUR_ID
  • Flight plans are parsed from XML and formatted into a Discord embed
  • The PDF button links directly to your SimBrief OFP (Operational Flight Plan)
  • PIREPs are posted to a designated channel (configurable by admins)
  • All CPAS flight data is logged for community tracking and progression
  • The system supports both fixed-wing and helicopter operations
  • Embed fields are dynamically populated based on your SimBrief data
