Piper Warrior II (PA-28-161) Checklist – MSFS 2020

EXTERIOR PREFLIGHT (WALK-AROUND)

Tie-downs / Chocks / Covers REMOVED
Fuel Quantity CHECK VISUALLY (Drain samples from both wings)
Fuel Caps SECURED
Oil Level CHECK (6–8 quarts, min 5 quarts)
Oil Cap / Dipstick SECURED
Control Surfaces FREE & CORRECT (Ailerons, Elevator, Rudder, Flaps)
Pitot Tube Cover REMOVED
Static Ports CLEAR
Tires / Brakes CHECK (Condition / Inflation)
Master Switch OFF during walk-around

COCKPIT PREPARATION

Parking Brake SET
Mixture IDLE CUTOFF
Throttle 1/2 INCH
Carb Heat OFF
Avionics Master OFF
Master Switch ON
Fuel Selector BOTH
Fuel Pump ON (Prime if cold engine)
Flaps UP
Trim TAKEOFF (Neutral or slightly nose up)
Circuit Breakers IN
Seat / Rudder Pedals ADJUSTED & LOCKED

ENGINE START

Area CLEAR
Mixture RICH
Throttle 1/4 INCH
Master / Ignition START (Monitor: Oil pressure rising within 30 sec, 1200 RPM)
Oil Pressure CHECK (Green arc: 60–90 psi, min 25 psi)
Fuel Pump OFF after start confirmed
Alternator CHECK (Ammeter charging, low voltage light out)
Engine Gauges MONITOR (CHT 150–400°F, Oil Temp 100–245°F)

RUN-UP AND MAG CHECK

Location RUN-UP AREA (Pointed into wind)
Parking Brake SET
Flight Controls FREE & CORRECT (Full movement check)
Throttle 1700–1800 RPM
Engine Instruments CHECK GREEN (Oil pressure 60–90 psi, Oil temp rising, CHT 300–350°F)
Magnetos CHECK (BOTH → L → BOTH → R → BOTH)
Max drop: 175 RPM
Difference L/R: < 50 RPM
Carb Heat CHECK (ON: expect 150–200 RPM drop, then OFF)
Vacuum / Suction Gauge CHECK (4.5–5.5 inHg)
Fuel Flow CHECK (Steady, no fluctuations)
Amps / Volts CHECK (Ammeter charging, voltmeter 13.5–14.5V)
Idle Mixture CHECK (Lean until rough, then enrich to smooth)
Throttle 1000–1200 RPM (After checks)

BEFORE TAXI

Flaps UP
Avionics Master ON (Allow 1 min for GPS initialization)
Transponder STANDBY (Mode C if available)
Radios / COM / NAV SET
Altimeter SET (Local field elevation)
Flight Instruments CHECK (AI, DG, VSI operational)
Flight Controls FREE & CORRECT (Full deflection check)
Brakes TEST (Light pressure, holding)

TAXI

Taxi Light ON (As required)
Fuel Pump ON for taxi on rough ground or low fuel
Engine Gauges MONITOR (Oil temp/pressure in green)
Mixture AS REQUIRED (Lean for taxi above 1000 ft density altitude)
Magnetos BOTH

BEFORE TAKEOFF

Parking Brake SET (At run-up/holding area)
Flight Controls FREE & CORRECT (Final check)
Flaps AS DESIRED (Typically 0–10° for normal takeoff)
Trim TAKEOFF (Neutral)
Fuel Selector BOTH
Fuel Pump ON
Mixture FULL RICH (Below 3000 ft MSL)
Carb Heat OFF
Flight Instruments SET (DG to compass)
Transponder ALT (If Mode C)
Briefing TAKEOFF / DEPARTURE / EMERGENCY

TAKEOFF

Area CLEAR (Final check of runway)
Flaps AS DESIRED
Fuel Pump ON
Throttle FULL (Smoothly, monitor RPM 2400–2500)
Engine Instruments MONITOR (Oil pressure 60–90 psi, CHT rising)
Airspeed ALIVE at 40 KIAS, rotate at 55–60 KIAS
Vy (Best Rate) 79 KIAS
Vx (Best Angle) 66 KIAS (with flaps up)

CLIMB

Fuel Pump OFF (Above 500 ft AGL, positive rate)
Flaps UP (If used, retract gradually)
Climb Speed 79–85 KIAS (Vy 79, cruise climb 85)
Mixture LEAN (Above 3000 ft MSL for best power)
Engine Gauges MONITOR (CHT ≤ 400°F, Oil Temp ≤ 245°F)

CRUISE

Power Setting 2300–2500 RPM (65–75% power)
Mixture LEAN (For peak EGT or per POH)
Cruise Speed 105–115 KIAS (75% power at 8000 ft)
Fuel Flow CHECK (8.5–9.5 GPH at 75% power)
Engine Gauges MONITOR (CHT 300–380°F, Oil Temp 180–220°F ideal)
Fuel Tanks BALANCE (Switch tanks hourly if not using BOTH)

DESCENT

Mixture RICH (Below 3000 ft MSL)
Carb Heat AS REQUIRED (ON when below green arc or visible moisture)
Descent Speed 100–120 KIAS (With 500–700 FPM descent)
Engine Cooling MONITOR (Keep CHT > 250°F during descent)
Fuel Pump ON (For landing)

BEFORE LANDING

Fuel Selector BOTH (Or fullest tank)
Fuel Pump ON
Mixture FULL RICH
Carb Heat ON (Below green arc)
Seat Belts SECURED
Door LATCHED
Flaps AS DESIRED (Typically 10° on downwind, 25–30° on final)
Approach Speed 65–75 KIAS (65 with flaps 30°, 75 with flaps 0°)
Trim SET for approach speed

LANDING

Final Approach 65–70 KIAS (With flaps)
Flare GENTLY (Power to idle, hold off)
Touchdown MAIN WHEELS FIRST
After Touchdown NOSE WHEEL DOWN (Gently)
Flaps UP (After clearing runway)
Carb Heat OFF

AFTER LANDING / TAXI-IN

Flaps UP
Transponder STANDBY
Lights AS REQUIRED
Engine Gauges MONITOR (During taxi)
Mixture LEAN for taxi if needed

ENGINE SHUTDOWN

Parking Brake SET
Avionics / Radios OFF
Fuel Pump OFF
Mixture IDLE CUTOFF (Smoothly pull to cutoff, engine stops)
Ignition OFF
Master Switch OFF

SECURING AIRCRAFT

Control Lock INSTALL (If available)
Pitot Cover INSTALL
Tie-downs SECURED
Fuel Selector OFF (If leaving aircraft)
Baggage / Documents REMOVED
Door LOCKED
