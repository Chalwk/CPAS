Airbus H125 (AS350 B3e)
Normal Checklist – MSFS 2020

EXTERIOR PREFLIGHT (WALK-AROUND)

Tie-downs / Rotor Brake REMOVED / DISENGAGED
Main Rotor Blades CHECK (Condition, freedom of movement, blade stops)
Tail Rotor CHECK (Condition, security, freedom)
Fuel Quantity CHECK VISUALLY (Drain sample, check for contamination)
Oil Level CHECK (Engine and MGB oil)
Skids / Landing Gear CHECK (Condition, security)
Pitot Tubes / Static Ports CLEAR / UNCOVERED
Cabin Doors / Windows SECURE / UNDAMAGED
Emergency Equipment VERIFIED (First aid, fire extinguisher)
Battery Master OFF during walk-around

COCKPIT PREPARATION

Parking Brake SET
Battery Master 1 & 2 ON
External Power CONNECTED (If available)
Fuel Shutoff Valve ON
Firewall Fuel Valve OPEN
Collective DOWN / LOCKED
Cyclic CENTERED
Rotor Brake OFF
Circuit Breakers CHECKED IN
Seats / Harnesses ADJUSTED & SECURED

AVIONICS POWER-UP

Avionics Master ON
VEMD (Vehicle & Engine Display) CHECK (Power-up sequence normal)
PFD / MFD INITIALIZE (Allow full boot)
Flight Instruments CHECK (Attitude, heading, altitude)
DC Voltage CHECK (≥ 23.5V)
Fuel Quantity CONFIRM (VEMD and visual)
Lamp Test PERFORM
Fire Detection Test PERFORM (Check audio/visual warning)

ENGINE START

Area CLEAR (Rotor arc 360° clear)
Master Warning / Caution CLEAR
Fuel Boost Pump ON
Engine Mode Selector FLIGHT IDLE
Start Button PRESS (Monitor: N1, N2, ITT rise)
Engine Parameters MONITOR:
• N1: Stabilize at 65–70%
• N2: Stabilize at 100%
• ITT: ≤ 850°C peak, stabilize ≤ 750°C
• Oil Pressure: Rise within 30 sec
Fuel Boost Pump OFF (After stable idle)
Generator CHECK (Online, voltage normal)

GROUND CHECKS & RUN-UP

Engine Warm-up 2–3 MINUTES at 65% N1 (Oil temp > 40°C)
Engine Parameters CHECK STABLE:
• Oil Pressure: 3.5–5.5 bar
• Oil Temp: 40–120°C
• ITT: ≤ 750°C
• Fuel Flow: Stable
Flight Controls FREE & CORRECT (Full deflection check)
Cyclic Friction ADJUST as desired
AFCS / Autopilot TEST (If installed)
Hydraulic System CHECK (Pressure normal, no leaks)
Rotor Brake VERIFY OFF

BEFORE TAKEOFF

Briefing TAKEOFF / DEPARTURE / EMERGENCY
Weight & Balance VERIFY (Within limits)
Performance Charts CONSULT (Hover ceiling, OGE/IGE)
Engine Mode Selector FLIGHT
Engine Parameters CHECK NORMAL:
• N1: 65–70%
• ITT: ≤ 750°C
• Oil Press/Temp: Green
Flight Instruments SET & CROSS-CHECK
Doors / Windows CLOSED & LATCHED
Seat Belts / Harnesses SECURED

HOVER CHECK

Lift to Hover SMOOTH COLLECTIVE to Nr 391–397 RPM
Engine Parameters MONITOR:
• Nr: 391–397 RPM
• N1: As required
• ITT: ≤ 750°C
• Torque: ≤ 100%
Vibrations CHECK NORMAL
Controls Response CHECK (All axes responsive)
Warning Lights NONE ILLUMINATED
Hydraulic Pressure CHECK NORMAL

AFTER TAKEOFF / CLIMB

Engine Parameters MONITOR (ITT, Torque, Ng within limits)
Climb Speed 70–80 KIAS (Best rate)
Vibrations CHECK (No unusual vibrations)
Warning Systems MONITOR (Master Caution/Warning)
Engine Instruments SCAN (Every 30–60 seconds)

CRUISE

Power Setting SET (Monitor torque ≤ 100%)
Cruise Speed 120–130 KIAS (Max cruise 140 KIAS)
Engine Parameters MONITOR:
• Nr: 391–397 RPM
• ITT: ≤ 750°C
• Torque: ≤ 100%
• Oil Temp/Press: Green
Fuel Management MONITOR (Consumption, balance)
Vibrations CONTINUOUS MONITOR
Cabin Temperature ADJUST as required

DESCENT / APPROACH

Power Reduction GRADUAL (Monitor ITT and torque)
Descent Speed 70–90 KIAS
Approach Briefing COMPLETE
Landing Area IDENTIFY & BRIEF
Engine Parameters MONITOR (ITT cooling rate)
Warning Systems CHECK CLEAR

BEFORE LANDING

Final Approach 60–70 KIAS (Adjust for conditions)
Landing Area FINAL CHECK (Clear, suitable)
Seat Belts SECURED
Doors VERIFY CLOSED
Engine Instruments CHECK NORMAL
Passenger Briefing COMPLETE (If applicable)

LANDING

Flare INITIATE at 30–50 ft AGL
Collective SMOOTH UP for cushion
Touchdown LEVEL SKIDS
After Touchdown COLLECTIVE DOWN
Rotor RPM STABILIZE at 65–70% N1

ENGINE SHUTDOWN

Cool-down 2–3 MINUTES at 65% N1 (ITT < 500°C)
Engine Mode Selector GROUND IDLE
Fuel Shutoff Valve CLOSE
Firewall Fuel Valve CLOSE
Engine Stop PERFORM (Monitor N1/N2 to 0%)
Avionics Master OFF
Battery Masters OFF

SECURING AIRCRAFT

Rotor Brake APPLY (Wait for Nr < 10%)
Tie-downs SECURE (Main rotor, tail rotor)
Blade Socks INSTALL (If required)
Pitot Covers INSTALL
Doors LOCKED
Flight Log COMPLETE
