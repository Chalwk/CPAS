Important Notes

  • Compatibility: Check each add-on's compatibility with your MSFS version
  • System Requirements: Some add-ons may have higher system requirements
  • Installation Order: Install scenery libraries (like NZA Model Library) before specific airports
  • Community Content: Free add-ons are community creations - support developers if you can
  • Not Required: You don't need all these add-ons to fly with CPAS - start with basics and add as you go
  • Support: Check developer websites/discords for installation help and troubleshooting

Getting Started Recommendations

If you're new to CPAS or flight simulation, start with:

  1. Essential: One aircraft from our fleet (R44 or Warrior for beginners)
  2. Navigation: SimBrief (free) for flight planning
  3. Scenery: NZCH Christchurch (our main base)
  4. Utility: Little Navmap (free moving map)
  5. Add later: PACX, FSRealistic, Navigraph for enhanced immersion

Remember: The most important add-on is your enthusiasm for flying!
